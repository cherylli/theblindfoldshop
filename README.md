## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Prerequisites

- npm (or yarn)

  ```sh
  npm install npm@latest -g
  ```

  ```sh
  npm install --global yarn
  ```
